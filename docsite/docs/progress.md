# Progress
Displays progress bar with an optional label.

## Props

### color _string_
Sets `$color`

### label _string_
The label for the bar

### max _number_
Sets the max value for the progress bar. Default is 1

### outline _bool_
Sets `$outline`

### row _bool_
Sets `$row`

### value _number_
Sets the value of the progress bar. Combined with max to determine how
filled the bar is.
