Flex

Container that uses flex layout by default with some nice default values.

Base

Windstorm Flex

Props

All windstorm functions are supported.

  • direction string

    Sets fl-dir

  • pad string

    Sets p

  • gap string

    Sets gap

  • cross string

    Sets fl-cr-a

  • main string

    Sets fl-m-a

Usage

<Flex cross="stretch">
    <Text>Content</Text>
    <Text>More content</Text>
    <Text>Event more content</Text>
<Flex>
