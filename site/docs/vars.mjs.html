vars Action

The vars action is used to set css variables on dom elements using a simple object to maintain the values instead of needing to do string interpolation and falsey checks manually.

Usage

//  Sets --screen-widt to 100% and unsets --other-thing
$: settings = {
    "screen-width": "100%",
    "other-thing": null,
}

<ws-screen use:vars={settings}>
</ws-screen>
