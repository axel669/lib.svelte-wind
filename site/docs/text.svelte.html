Text

Simple component with some common props for types of text.

Props

All windstorm functions are supported.

  • adorn bool

    Sets $adorn

  • block bool

    Sets block

  • subtitle bool

    Sets $subtitle

  • title bool

    Sets $title

Usage

<Text>Some text</Text>
<Text title>Bigger text</Text>
<Text subtitle>Smaller text</Text>
<Text block>Text on its own line</Text>
