Button

Basic clickable component for handling user interaction.

Base

Windstorm Button

Props

All windstorm functions are supported.

  • color string

    Sets $color

  • compact bool

    Sets $compact

  • for string

    The ID of a dom element that the button is tied to. Only works with label buttons.

  • label string

    If true, the button will be displayed in the DOM using a

  • variant string

    Sets the button type to @<variant> (ex: "flat" -> @flat). Default variant is "flat". Currently supports "flat", "fill", "outline".

Events

  • click

Example

<Button on:click={action}>
    Flat Button
</Button>
<Button on:click={action} variant="outline">
    Outlined Button
</Button>
<Button on:click={action} variant="fill" color="danger">
    Filled Button
</Button>

<Button label for="thing">
    Toggle
</Button>
<input type="checkbox" id="toggle" />
